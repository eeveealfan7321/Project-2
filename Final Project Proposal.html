Project Concept

Goal, Object and Audience of Website

The overall goal of the project is to create a easy to read and follow tutorial on how to change outfit and texture on booth models in unity. This is to help user who maybe are new to unity and the Vrchat Creator Comapnsion. The object of the website is to (1) Create a as mention easy to read and follow tutorial on hwo to change the outifts and texture of a booth model, and how the VCC works.(2) Provide all the add on and extra files that are need to ensure the upload of the model goes smoothly. (3) Have a video tutorial that provides a more visual tutorial to futher help user with the unity interface.
